# bd_loadingscreen

# How to Install
- Download file and drop in your “**resources**” folder
- Add to your **server.cfg** file:  ***ensure bd_loadingscreen***
- Install done. Have Fun !!

# Controls
- **Space** (Play/Stop music)
- **ArrowUp-Key** (Volume up)
- **ArrowDown-Key** (Volume down)
